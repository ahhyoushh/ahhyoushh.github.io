Why no-cursor?

100% Local
Runs entirely on your machine. Zero cloud dependencies. No external API calls. You own your data.
Deterministic
Same input, same output. Reproducible across runs with local GGUF models. Predictable behavior.
Diff-Based Edits
All changes shown as unified diffs. Review before apply. No silent mutations or surprises.
File Safety
Content hashing prevents corruption. Automatic backups. Explicit approval required for all edits.
Single File Focus
One file at a time. Simple, focused, and verifiable. No complex multi-file refactoring.
No Auto-Mutations
You decide when changes commit. Dry-run mode lets you preview without changing files.
